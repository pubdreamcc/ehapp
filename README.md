# 移动端项目脚手架
## ehapp 脚手架使用说明

1. 全局安装：`npm install ehapp -g`

2. 新创建一个app项目：`ehapp init test-app`，如需引入react hooks类型校验及hook语法封装lib，可带参数 `-hook` eg: `ehapp init test-app -hook`

> app 项目命名规范：单独英文或英文 + `-` 命名方式

3. 业务开发，打包，将文件拖入zapp repo 统一托管
